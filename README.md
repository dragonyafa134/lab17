### 4. Реализация алгоритмов на языке C#
*   **Линейные алгоритмы:** Последовательное выполнение команд.
*   **Ветвления:** Использование конструкций `if`, `if-else` и множественного выбора `if-else if-else` для принятия решений.
*   **Циклы:** Организация повторяющихся действий с помощью циклов `while` и `for`.
*   **Работа с переменными:** Использование переменных различных типов данных (`int`, `double`, `string`, `bool`) для хранения и обработки информации.
*   **Массивы:** Хранение и обработка наборов однотипных данных.
